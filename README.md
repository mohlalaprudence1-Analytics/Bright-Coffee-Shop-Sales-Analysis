### Bright-Coffee-Shop-Sales-Analysis
This project analyzes six months of transactional data from Bright Coffee Shop to support business decision-making for a newly appointed CEO. The goal of the analysis is to understand revenue performance, identify key product drivers, and uncover customer purchasing patterns across time and store locations.

### Problem Statement
The coffee shop has been under performing in the last 6 months, I am tasked to perform diagnostic on what caused the drop in revenue.

### Aim
The aim of this project is to provide analysis with key insights on the revenue decline

### Objectives
- Identify key revenue drivers
- Understand customer purchasing behavior
- Analyze product and store performance
- Provide data-driven recommendations

### Methodology
1. Project Planning
- Mindmap (Miro) to structure the analysis approach (Process and Data Mindmap)
- Gantt Chart to outline project phases and timelines

- 2. Data Processing (Databricks SQL)
- Cleaned and transformed raw CSV data
- Created revenue based oncalculations  
Aggregated data by:
- Product categories
- Time intervals
- Store locations
  
3. Data Analysis
  - Converted data from csv to excel file
  - created revenue column
  - Built Pivot Tables

4. Visualization
   - Developed charts and graphs using pivot tables
   - Created a Presentation using canva theme to powerpoint
  
   ### Key Insights
   - Sales peak in the afternoon (12:00–15:00)
   - Coffee and tea are the main revenue drivers
   - Revenue is concentrated in a small group of top products
   - Hell’s Kitchen is the top performing store
   - Sales are generally higher on weekends

     ### Project Structure
     - Project Description & Raw Data – Dataset and case study brief
     - Project Planning – Miro flowchart & Gantt chart
     - Data Processing – SQL scripts
     - Data Analysis – Excel pivots & charts
     - Presentation & Dashboard – Final outputs
  
   ### Conclusion
Revenue performance is driven by time-based demand, strong product preferences, and store-level differences. By focusing on peak periods, top products, and targeted strategies, the business can improve performance and drive sustainable growth.
